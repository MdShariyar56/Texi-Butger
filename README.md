# 🚖 Taxi Kitchen - Digital Menu Card

A modern **Restaurant Menu Web App** built with HTML, Tailwind CSS (DaisyUI), and JavaScript.  
It fetches food data from a REST API and allows users to browse categories, view food details, and add items to cart.

---

##  Live Demo
  https://texi-butger.vercel.app/

---

##  Features

-  Dynamic food categories
-  Food listing from API
-  Food details modal (YouTube video support)
-  Add to cart functionality
-  Quantity update system
-  Auto total price calculation
-  Fast and responsive UI
-  Built with Tailwind + DaisyUI

---

##  Tech Stack

- HTML5
- Tailwind CSS + DaisyUI
- JavaScript (ES6)
- Fetch API
- Vercel (Deployment)
